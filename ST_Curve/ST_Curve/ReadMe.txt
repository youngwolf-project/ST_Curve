========================================================================
    ACTIVEX 控件 DLL : ST_Curve 项目概述
========================================================================

控件向导已为 ST_Curve ActiveX 控件 DLL 创建此项目，其中包含一个控件。

此主干项目不仅演示了编写 ActiveX 
控件的基本方法，而且还可以作为编写控件特定功能的起点。

本文件概要介绍组成 ST_Curve ActiveX 控件 DLL 的每个文件的内容。

ST_Curve.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，
    其中包含生成该文件的 Visual C++ 
    的版本信息，以及有关使用应用程序向导选择的平台、
    配置和项目功能的信息。

ST_Curve.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。 
    它包含有关项目文件与筛选器之间的关联信息。 在 IDE 
    中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。
    例如，“.cpp”文件与“源文件”筛选器关联。

ST_Curve.h
    这是 ActiveX 控件 DLL 的主包含文件。  其中包括其他项目特定的文件，如 
    resource.h。

ST_Curve.cpp
    这是包含 DLL 初始化、终止和其他簿记的代码的主源文件。

ST_Curve.rc
    这是项目使用的 Microsoft Windows 资源的清单。
    此文件可以直接用 Visual C++ 资源编辑器进行编辑。

ST_Curve.def
    此文件包含关于在 Microsoft Windows 上运行所必需的 ActiveX 控件 DLL 的有关信息。

ST_Curve.idl
    此文件包含控件类型库的对象描述语言源代码。

/////////////////////////////////////////////////////////////////////////////
CST_CurveCtrl 控件：

ST_CurveCtrl.h
    此文件包含 CST_CurveCtrl C++ 类的声明。

ST_CurveCtrl.cpp
    此文件包含 CST_CurveCtrl C++ 类的实现。

ST_CurvePropPage.h
    此文件包含 CST_CurvePropPage C++ 类的声明。

ST_CurvePropPage.cpp
    此文件包含 CST_CurvePropPage C++ 类的实现。

CST_CurveCtrl.bmp
    此文件包含一个位图，当 CST_CurveCtrl 
    控件出现在工具面板上时，容器将使用此位图来表示该控件。  此位图包含在主资源文件 
    ST_Curve.rc 中。

/////////////////////////////////////////////////////////////////////////////
其他标准文件：

stdAfx.h，stdAfx.cpp
    这些文件用于生成名为 ST_Curve.pch 的预编译头 (PCH) 文件和名为 
    StdAfx.obj 的预编译类型 (PCT) 文件。

resource.h
    这是标准头文件，可用于定义新的资源 ID。
    Visual C++ 资源编辑器读取并更新此文件。

其他注释：

控件向导使用“TODO:”来指示应添加或自定义的源代码部分。
